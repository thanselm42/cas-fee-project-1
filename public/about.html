Toby's Note App

Introduction

Need to get order in your mess of ToDo-lists and post-its? This is your app! Organize your notes by importance, due date and more. Or set a color label to group similar notes.

To speed up your work, use the key-bindings described in the side-text on the right.

This app was made as part of the CAS-FEE Project-1 at OST.

Check out the source-code on GitHub.

Firefox-Issues

Some features do currently not work properly in FireFox. Please use Chrome instead!

If you like to use FireFox enable the following properties in about:config

  • layout.css.backdrop-filter.enabled
  • gfx.webrender.all
  • layout.css.backdrop-filter.enabled

Unfortunately the date-time chooser is still not working in FireFox-Desktop (v.89 and 90.0b4).

You can try to enable the following properties in about:config (may be this bug will be fixed in a newer version)

  • dom.forms.datetime-local
  • dom.forms.datetime-local.widget

Statistics and Administration

Quick edit key bindings

List-View

Press + key to add a new note.

Press the space key to hide/show completed notes.

Edit-View

Press the esc key to cancel and close the edit-view.

Press ctrl+s to save the current note.

Press ctrl+shift+s to save the current note and close the edit-view.

Press the ctrl+← key to cancel the current note and open the previous one.

Press the ctrl+→ key to cancel the current note and open the next one.
